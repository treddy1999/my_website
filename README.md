## ğŸš€ Getting Started

### Prerequisites

* Node.js 18+
* npm or yarn

### Installation

1. **Clone the repository**  
```bash
git clone https://github.com/treddy1999/portfolio-website.git
cd portfolio-website
```
2. **Install dependencies**  
```bash
npm install
# or
yarn install
```
3. **Run the development server**  
```bash
npm run dev
# or
yarn dev
```
4. **Open your browser**Navigate to <http://localhost:3000>
